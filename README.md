# vislib

robotics c++ template library

## version: 0.0.23

## Description

A "vislib_robo" library includes abstract classes or interfaces for control of some device types + some controllers and abstractions related to robotics concepts.

## Installation

vislib_robo can be either manually added to project or installed with platformio package manager by adding "alivka/vislib_robo" to lib_deps field of platformio.ini file, e.g.:

```ini
[env]
platform = ...
board = ...
framework = ...
lib_deps =
    alivka/vislib_robo
    ...
```
